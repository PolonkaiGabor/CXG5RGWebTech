Github használatának legfontosabb parancsai:

1. apt-get install git-core git-doc git-gui gitk

2. git help git help parancs # pl git help commit

3. Új git repository létrehozása git init

4. Változások listázása git status

5.File hozzáadása, áthelyezése, törlése git add filename git mv oldfilename newfilename git rm filename

6. Változások mentése git commit git commit -m 'commit message' git commit -a # staging area kihagyása

7. Commitok listázása git log git log -p # lista diff-el
